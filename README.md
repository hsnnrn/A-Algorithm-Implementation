# A-Algorithm-Implementation
In this project, we implement the A* algorithm for finding shortest paths.

This repository contains an implementation of the A* (A Star) algorithm in Python. A* is a popular pathfinding and graph traversal algorithm, widely used in various applications such as robotics, video games, and navigation systems.
Usage
Clone the repository to your local machine
Navigate to the project directory
Run install pygame : pip install pygame
Debug.
